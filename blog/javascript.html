Seventh Post on Ruby Loops vs Javascript Loops

2/12/2016

Loops

Programming languages have many ways of iterating through data, and there are many common ones. Different programs have multiple ways of doing the same thing, while others are more elemental about the process. Ruby is in the latter case, were its philosophy is to have as many different ways to do the same thing as possible, whiel Javascript is more of the former, staying to the basics.

Ruby

There are many methods to iterate through our data in Ruby. Just like there are many ways of tackling a problem, there are many tools at our disposal. Among the most frequent uses are for ... in ..., .each, .map, and while. And ieach of them have a variation depending on the task at hand, for example if we are looping through a hash, or an array, if we want keys only or values for a hash, if we want to reference our approach based on the contents of the elements inside an array or the location where the data resides in the array. We can see it in action below.

Javascript

Javascript is influenced syntactically by Java and C/C++ languages. It is evident from the way we state functions, loops, and even terminate lines of code (using semicolons). So to iterate through our data, we can intrust ourselves to the basic options of for, while, map, and forEach. When iterating in Javascript, we want to think of it as the essential tools needed to build great things, we will only be limited by our imagination. So let's take a look with the examples below.

Examples


Ruby Iterations

# Array used for the examples
food = ["pizza", "burger", "hot dog", "nachos"]

#using the for ... in method
for i in food
    puts i
end

##result 
pizza
burger
hot dog
nachos
 => ["pizza", "burger", "hot dog", "nachos"]
 
#using the each method
food.each {|item| puts item}

##result
pizza
burger
hot dog
nachos
 => ["pizza", "burger", "hot dog", "nachos"]     

#using the map method
food.map {|item| puts item}

##result
pizza
burger
hot dog
nachos
 => [nil, nil, nil, nil]
 
#using while method
i = 0
while (i < food.length)
    puts food[i]
    i += 1
end

##result
pizza
burger
hot dog
nachos
 => nil

Javascript Iterations

# Array used for the examples
var food = ["pizza", "burger", "hot dog", "nachos"]

#using the for ... in method
for (var i = 0; i < food.length; i++) {
    console.log(food[i])
}

##result 
pizza
burger
hot dog
nachos
=> undefined
 
#using the each method
food.each {|item| puts item}

##result
pizza
burger
hot dog
nachos
 => ["pizza", "burger", "hot dog", "nachos"]     

#using the map method
food.forEach(function(item) {console.log(item)})

##result
pizza
burger
hot dog
nachos
=> undefined
 
#using while method
var i = 0
while (i < food.length) {
    console.log(food[i])
    i += 1
}

##result
pizza
burger
hot dog
nachos
=> 4
